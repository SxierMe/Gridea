WinSnap - 一个值得你下载的截图软件

2023-04-28 Views 效率349字2 min read繁

WinSnap 是一个非常好用的截图软件，可以让你的截图更具美感。

如果你经常写教程类的博文，那么一个好用的截图工具一定是必不可少的。那么Windows上如何像 macOS 一样为图片添加阴影效果呢？

好用的截图工具非常多，如果要找 Windows 上最好用的免费截图标注工具 ，那么 Snipaste 一定当之无愧。不过今天的主角并不是它，而是 Snipaste 至今还未支持的截图中的一个细节—— 截图四周的阴影 。

一、WinSnap

WinSnap 是一款功能全面的 Windows 截图工具，支持多种截图方式，使用起来方便快捷。通过自定义快捷键，让你的接入效率更上一层楼。
