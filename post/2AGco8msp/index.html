一、Rime 简介

Rime 是一个最强的中文输入法引擎。它支持各种各样的形码、还有其他我叫不出名字总之很牛逼的打字方式。它本身是一个输入法引擎，内置了一个朙月拼音简体字的输入法，能用但总有一种软绵绵的感觉你知道吧。 还有更牛逼的是它能多端平台使用，而且还能

所以本次我们用Rime中文输入法引擎+第三方优秀的拼音输入方案（rime-ice）来练成每次在键盘敲击的时候会展现出气势磅礴的感觉。

废话不多我们直接开始。

GitHub项目地址：https://github.com/rime

RIME官方主页：https://rime.im/

二、安装教程

2.1 基本要求

  • 操作系统 Windows 系统
  • 有条件就将原先的Rime备份了再卸载
  • 懂 Windows 基本操作
  • 懂中文且认识26个字母

2.2 Rime输入法引擎安装

💡 Rime的安装路径一定不要带中文的，建议直接选择默认安装就行。

  • 打开主页后找到并点击下载 小狼毫 0.15.0

  • 下载到本地后双击进行安装

  • 安装路径我个人强烈建议是默认的安装路径，因为安装路径中只要有中文你就无法使用简体。

  • 默认点击安装即可

  • 安装完成，此时要稍等一会它会自动在后台部署。

  • 此时默认的输入法方案是朙月拼音- 简体字，你可以在此选择你自己想要的输入法方案。不过并不用着急，因为我们要安装雾凇拼音的输入法方案

2.3 雾凇拼音安装切换

雾凇拼音（官方名称叫rime-ice）是基于Rime中文输入法引擎做的优秀输入法，搭配Rime输入法引擎真正使用起来和搜狗、微信键盘不输于他们，可谓是开箱即用啊。另外作者的更新超级肝，因为这个输入法他自己也在用，所以大家都可以搭这趟顺风车了。

甚至输入法配色方面真的有一说一可以和张小龙同学的小而美掰头掰头了。

GitHub项目地址：https://github.com/iDvel/rime-ice

官网地址：https://dvel.me/posts/rime-ice/

（雾凇拼音使用官网）从官方展示的图片来看，真的非常强大👍

雾凇拼音加配色

  • 进入雾凇拼音GitHub项目 找到并点击左上角Code → download ZIP ，下载整个项目文件

  • 本地进行解压缩，进入解压缩的文件夹里边，Ctrl + A 进行全选，Ctrl + C进行复制

  • 进入Rime用户配置文件夹 %APPDATA%\Rime 复制之后即可进入，粘贴到用户配置文件夹里边

  • 粘贴完成后，不管用什么办法，找到小狼毫的重新部署，点击后稍等几分钟。 这期间它会不断的重新部署，电脑会偶尔提示小狼毫维护中或是小狼毫部署中等等，约会有5次左右。

  • 觉得差不多了， 按一下F4 ，方向键找到雾凇拼音回车就行。

  • 此时你就可以开心的使用了

  • 还有更好玩的操作等你来挖掘

以上就是超级简单的安装和配置方案了。 当然，高度可定制化的东西怎么可能这么简单。如果有需要的可以继续往下继续看。如觉得配置好就行的可以就此别过了。
