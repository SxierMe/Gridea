[2024]亲测使用iPhone美区Apple ID 开通或续费 ChatGPT Plus

· 2023-09-06 · # 技术

ChatGPT相信大家都不陌生了，作者因工作需要ChatGPT Plus，但受限于手头没有外币卡，无法绑定或是付费。淘宝代付虽说可以，但终归是不够安全(手续费太高了...)。

简单介绍

本篇教程一共分为两个部分。
第一：使用支付宝给美区Apple ID 充值礼品卡
第二：使用Apple ID 缴费开通或续费ChatGPT Plus

工作准备

  • 正常使用的支付宝
  • 使用过一段时间的或是正常使用的美区Apple ID 并提前在App Store登录好(新注册的可能会触发风控）。
  • 提前在iPhone上ChatGPT IOS - 下载地址

支付宝充值20U

  1. 打开支付宝，首页搜索栏搜索 PockytShop，点击进去注册。最好用自己的正常使用邮箱，不要用临时邮箱。

  2. 下滑找到App Store & iTunes US并点击，在卡片底下找到 $2-500并点击，输入20，点击下方的立即购买。正常的付款就行了。

  3. 付款完成后， 稍等片刻就跳转到他的订单首页，会出现一个兑换码，点击复制。

  4. 打开App Store，首页右上角点击头像。找到并点击 兑换充值卡或代码。

  5. 点击选择 手动输入兑换码 ，黏贴、点击兑换。稍等片刻就能在你的美区Apple账号充值成功20U的余额了。

ChatGPT Plus开通或续费

  1. 打开ChatGPT App，右上角点击 三个点，点击进入 Settings
  2. 找到并点击 Upgrade to ChatGPT Plus, 按照提示开通付款即可。

注：因为开通操作的时候没有截图记录， 所以复现的时候就少了一些图片的操作了。
